# 🎮 27 Offline Games

A collection of 27 browser-based games built with pure HTML, CSS, and JavaScript.
No installation, no internet, no ads, no tracking — just open and play.

---

## 📲 Download Android App

[![Download APK](https://img.shields.io/badge/Download%20APK-Latest%20Release-brightgreen?style=for-the-badge&logo=android)](https://github.com/wild-coder-ux/27-offline-games/releases/latest)

> **Installation:** Settings → Security → enable **Install from unknown sources** → open the APK.

The Android app is a lightweight **WebView wrapper** — all 27 games are bundled locally inside the app. No browser needed, no internet required, works fully offline after install.

## ✨ Features

- 🔌 **100% Offline** — no internet connection required
- 🚫 **No Ads** — ever. Zero interruptions.
- 🔒 **No Tracking** — zero analytics, zero data collection
- 📱 **Mobile Friendly** — built for Android, works in any browser too
- ⚡ **Instant Play** — no sign-in, no setup, just tap and play
- 🧩 **Pure HTML/CSS/JS** — no frameworks, no dependencies

---

## 🛠️ Tech Stack

- Games: Pure **HTML5**, **CSS3**, **JavaScript** — each game is a self-contained `.html` file
- Android app: **WebView wrapper** — bundles all games as local assets, zero network calls
- Shared logic via `common.js`
